# 🛡️ ACLR Cyber Threat Detection System

An AI-powered Cyber Threat Detection System built using Machine Learning, Deep Learning, TensorFlow, and Streamlit.

The application helps detect various cyber threats including malware, malicious URLs, suspicious IP addresses, and potentially dangerous files through an interactive web interface.

---

## Features

- Malware Detection
- URL Threat Detection
- IP Threat Detection
- File Threat Detection
- Manual Threat Analysis
- Automatic Threat Scan
- Interactive Streamlit Dashboard
- Machine Learning & Deep Learning Models

## Installation

Clone the repository

```bash
git clone https://github.com/Pranith-g/ACLR-Cyber-Threat-Detection-System.git
```

Move into the project

```bash
cd ACLR-Cyber-Threat-Detection-System
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
